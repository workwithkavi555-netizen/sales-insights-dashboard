Sales Insights Dashboard using Excel & Power BI

This project provides a comprehensive Sales Insights Dashboard built using Excel and Power BI to analyze key business metrics.
It helps in identifying trends, forecasting sales, and understanding high-performing products and regions.

 Key Features
	•	Data cleaning & transformation
	•	Interactive visualizations
	•	Sales trend analysis
	•	Region-wise performance
	•	Top products & revenue insights

 Project Files
	•	sales_data_sample.csv — Raw sales data
	•	sales_insights.xlsx — Processed Excel file
	•	sales_placeholder.png — Dashboard preview
	•	README_powerbi.md — Steps to build dashboard
